Getting Started

Follow this guide to initialize the Pixora rendering pipeline and bootstrap your first high-performance graphics application.

01

System Requirements

Hardware

Vulkan 1.2+ capable GPU or Apple M-series

Software

Node.js 18.x or Rust 1.70+

02

Installation

Use our CLI tool to scaffold a new project with all the necessary shader configurations and build hooks.

Terminal
$ npm install -g @pixora/cli
$ pixora init my-renderer
03

First Rendering Pipeline

The core of Pixora is the Pipeline object. Create a basic scene by defining your entry point in index.px.

src/index.px
// Initialize engine context
import { Scene, Camera } from '@pixora/core';
const scene = new Scene({ antialiasing: true });
const camera = new Camera({ fov: 75 });
// Mount to DOM
scene.mount(document.querySelector('#viewport'));
scene.onRender(() => {
camera.rotate(0.01);
});
