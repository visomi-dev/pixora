Users & Identity

GET /v1/users/{user_id}

Retrieves the details of an existing user. You need only the unique user identifier that was returned upon creation.

Parameters

Name Type Description
user_id string The unique identifier for the user.
include_metadata boolean Optionally include extended metadata fields.

Sample Response

application/json 
{
  "id": "usr_94210",
  "object": "user",
  "name": "Jane Doe",
  "email": "jane@pixora.io",
  "created_at": 1672531200,
  "metadata": {
    "tier": "pro",
    "region": "us-east"
  }
}
POST /v1/users

Creates a new user object within your workspace. Requires a valid API key with administrative permissions.

Request Body

Field Type Required
email string check_circle
name string check_circle
avatar_url string —

cURL Example

curl https://api.pixora.io/v1/users \
  -u <test_api_key>: \
  -d email="newuser@example.com" \
  -d name="Alex Smith"
